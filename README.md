# validaTortilla
A spicy Validator Library for Android.

## Bevezető

Validálni mindent is kell. 


## Kezdő lépések

Vegyük fel a könyvtárat függőségként a modul szintű `build.gradle` fájlban:

```groovy
dependencies {
    implementation 'com.github.gerfalcon:validatortilla:1.0.0'
}
```
